## Featured Projects

### [COUNTER-HATE: Multimodal Counterfactual Augmentation for Bias-Aware Hate Detection](https://github.com/VeerajSai/HateSpeech-FairCF-Counterfactual-Dataset-for-Bias)

Multimodal hate speech detection framework using LLM-generated counterfactual text rewrites and synthetic images for fairness-aware training. The pipeline combines adversarial training, counterfactual augmentation, and multimodal fusion to improve model robustness and reduce demographic bias.

**Tech:** Python, PyTorch, Hugging Face Transformers, HateBERT, CLIP ViT-B/32, EfficientNet-B0, TF-IDF, scikit-learn

---

### Greenko Wind Power Scheduling

Time-series forecasting system for renewable energy scheduling across 30+ wind plants, operating at 15-minute resolution. Built a Ridge + LightGBM ensemble pipeline that reduced scheduling MAE from 7.28 MWh to 3.70 MWh. Includes a Streamlit operator dashboard and a local Ollama-based RAG chatbot for on-site query support.

**Tech:** Python, LightGBM, Ridge Regression, Pandas, Streamlit, Ollama

---

### [CryptoStats](https://github.com/VeerajSai/CryptoStats)

A KDE Plasma 6 widget that displays real-time cryptocurrency pricing on the Linux desktop. Built using QML with live data fetched via curl, designed for minimal overhead and clean integration with the KDE Plasma environment.

**Tech:** QML, Bash/curl, Linux, KDE Plasma 6

---

### [BCI-MI: EEG Motor Imagery Classification](https://github.com/VeerajSai/BCI-MI-using-CNN-LSTM)

Experimental research project exploring EEG-based motor imagery classification for brain-computer interface (BCI) applications. Uses a CNN+LSTM architecture for spatial and temporal feature extraction from EEG signals. This is work in progress — the repository is currently being refactored to improve evaluation reliability and address potential data leakage in the preprocessing pipeline. Not a finalized or production-validated model.

**Tech:** Python, PyTorch, CNN/LSTM deep learning, EEG signal preprocessing

---

### [SemShift](https://github.com/VeerajSai/SemShift)

Detects semantic shifts, claim changes, tone drift, and risk changes across versions of text documents and prompts. Useful for tracking LLM prompt evolution, document revisions, and policy change analysis.

**Tech:** Python, NLP, LLM workflows
